# Glossary

Términos del dominio del framework `agex`. Una línea por entrada.

## Skills

- **Skill vertical**: skill asociada a una fase concreta del flujo SDD
  (p.ej. `close-requirement` en `/sdd-spec`). Ver ADR-0001.
- **Skill horizontal**: skill reutilizable por varios comandos del
  flujo (p.ej. `collaborative-decision`). Ver ADR-0001.
